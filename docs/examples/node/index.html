Node.js Example Suite

Use these scripts to exercise Helios Network directly from Node 18+ without bundlers.

Prerequisites

  • Install dependencies with npm install.
  • Build the WASM artefacts so the dist/ bundle is current: 
    npm run build
npm run build:wasm
  • Ensure you are running on Node.js 18 or newer (the package targets native ES modules).

Run the Samples

Execute the scripts from the project root. Each script logs its operations to stdout, so you can follow the graph mutations in real time.

node docs/examples/node/basic-usage/main.mjs
node docs/examples/node/attributes/main.mjs
node docs/examples/node/iteration/main.mjs
node docs/examples/node/modifying-graphs/main.mjs
node docs/examples/node/saving-loading/main.mjs
  • Basic usage — initialize the network and inspect neighbours.
  • Attributes — register numeric/JS-backed stores.
  • Iteration — work with selectors and traversal helpers.
  • Modifying graphs — remove entities and optionally compact.
  • Saving & loading — round-trip serialized payloads.
