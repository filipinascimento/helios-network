Node.js Example Suite

Use these scripts to exercise Helios Network directly from Node 18+ without bundlers.

Prerequisites

  • Install dependencies with npm install.
  • Build the WASM artefacts so the dist/ bundle is current: 
    npm run build
npm run build:wasm
  • Ensure you are running on Node.js 18 or newer (the package targets native ES modules).

Run the Samples

Execute the scripts from the project root. Each script logs its operations to stdout, so you can observe the mutations Helios performs.

node docs/examples/node/run.mjs
node docs/examples/node/attributes.mjs
  • run.mjs creates a small undirected network, adds nodes and edges, and then disposes of the instance.
  • attributes.mjs showcases attribute registration across every scope, including multi-dimensional and JavaScript-backed values.
